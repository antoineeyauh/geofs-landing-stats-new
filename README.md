# Geofs Landing Stats Addon
Get the data that matters. This addon provides a clean, top-left overlay immediately upon touchdown. It features:
1. Real-time FPM (Vertical Speed)
2. True Airspeed at impact
3. Automatic Aircraft & City detection
4. Bounce Tracking
# To add this addon
1. Copy the code in the main.js section
2. Paste it into tampermonkey
You're all set!
